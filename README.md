## 1. Prosjektidé og problemstilling

### Beskrivelse
```markdown
- Dette prosjektet er en type digital piano. 
- Det er mulig å bruke tastatur til å spille av lyd som et keyboard.

Spørsmål og problem:
"Hvilket problem løser løsningen?"
- Jeg lærer å lage et digitalt/simulert keyboard
"Hvem er målgruppen?"
- Meg selv, og folk som er interreserte i slik.
```


**Installasjon og kjøring av prosjektet:**

1. **Klon prosjektet:**
```markdown
    git clone https://github.com/SD-Shar/piano_thing.git
```
2. **Installasjons krav:**
    HTML, CSS og Javascript


3. **Kjøring av prosjekt:**
   Trykk på "Go Live" på HTML siden

---

## 2. Funksjonalitet til prosjektet

1. Piano kan identifisere hvilken pianonøkkel blir valgt
2. Tastatur på laptpop blir linket med pianoet og pianonøkkel blir identifisert
3. Lyd på piano spilles av med "keydown" funksjon

## 5. Refleksjon

**Problemer jeg møtte**
 ```markdown
  - Jeg klarte ikke identifisere den enkelte nøkkelen på keyboardet
  og brukte ganske lang tid på å finne ut av hvordan jeg skulle fikse det.
  Til slutt gav jeg en id til hver nøkkel siden keyboardet mitt var ganske kort,
  og brukte if (event.key === '[id]') så det ville spille av lyd
  og jeg brukte innercss for å style/vise at den er valgt.
 
 ```
